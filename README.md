# boss-distill
Chat Distill 是 Prompt 工程技能包：滤掉群聊寒暄与跑题噪音，从对话、语音转文字与截图中提取需求约束，编译成六柱 Prompt，把碎片讨论变成可落地方案。Chat Distill is a prompt-engineering skill that strips chat noise, pulls needs and constraints from chats, transcripts, and screenshots, and compiles them into a six-pillar Prompt, turning fragmented talk into actionable specs.
